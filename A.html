A — Languages and Repositories

Due Friday, September 9, 9:59pm

Purpose read assignment descriptions and react to them appropriately; experiment with your git repositories; live up to task specifications

Each partner (assuming you have one, alone if you do not) should submit a file. That file should be named username.md, where "username" is your Atlassian account name. Each file should have two lines on it. The first line should be your preferred programming language/environment for working with large software systems. The second line should be your backup choice, if for some reason the management at your organization rejected your first choice.

Neither of these need to be the language in which your partnership ultimately end up working; this should however let me know in what languages you think you would feel most comfortable.

Delivery You must deliver these two files in a directory called A in your repository (master branch).
